Backend
Функция make_prices_in_periods_objects_from_data_gov_ru_data_file(path) преобразует исходные данные в удобный для дальнейшей работы вид
Далее функции с самодокументируемыми именами возвращают соответствующие значения
В случае, если входные данные за пределами данных в исходном файле, возвращается ноль
Ведётся логгирование log_it() сохраняет данные о времени вызова функций и продолжительности их выполнения в файл log.txt)
Предусмотрена возможность кеширования данных для запросов, cach_it проверяет запрос с данными, и если такой же был, берёт данные из файла each.txt, если нет, выполняется функция и результат добавляется в файл)
Frontend
Отображаются данные в пределах допустимых значений из исходного файла
В случае, когда нужна одна дата как данные для запроса, второе поля ввода скрывается
Пользуюсь тем, что лишние данные для запроса к серверу не вызывают исключений, вся работа делается в одной строке, всегда передаётся название метода и две даты, даже, когда нужна одна дата или даты не нужны
Вывод лога - новые данные сверху
